VNN Uniform Style Guide

Filters

  1. formatTeamName
  2. fullName
  3. ordinalSuffix
  4. phoneNumber
  5. shortLevelName
  6. stripHighSchool

formatTeamName

Formats the supplied Team resource as gender level sport.name.

Given the Team resource: 

{
  gender: 'Boys',
  level: 'Varsity',
  sport: {
      name: 'Dumpster Diving'
  }
}

The following filter:

{{team | formatTeamName}}

Will produce:

Boys Varsity Dumpster Diving

fullName

Formats the supplied Name object as [firstName] [middleName] [lastName] with all pieces omitted if not present or null.

Given the User resource: 

{
  id: 5,
  name: {
    first: 'Dexter',
    last: 'Morgan'
  }
}

The following filter:

{{user.name | fullName}}

Will produce:

Dexter Morgan

ordinalSuffix

Appends an ordinal suffix to a number.

The following filter:

{{'100' | ordinalSuffix}}

Will produce:

100th

phoneNumber

Formats a phone number as (###) ###-#### 

{{'1234567890' | phoneNumber}}

Will produce:

(123) 456-7890

shortLevelName

Reduces an occurrence of Junior Varsity in a string to JV.

The following filter:

{{'Junior Varsity Golf | shortLevelName}}

Will produce:

JV Golf

stripHighSchool

Strips the supplied School resource of High School when it appears at the end of the string.

Given the School resource: 

{
  name: 'Knibb High School'
}

The following filter:

{{school | stripHighSchool}}

Will produce:

Knibb High School

Given the School resource: 

{
  name: 'Knibb High School for the Awesomely Gifted'
}

The following filter:

{{school | stripHighSchool}}

Will produce:

Knibb High School for the Awesomely Gifted
